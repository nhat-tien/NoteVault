## Các bước thực hiện
1. Mở file text
2. Nhập nội dung bên dưới

```txt
[InternetShortcut] 
URL=obsidian://{* Obsidian URI *}
```
 
 3. Lưu file bằng đuôi `.url`
 4. Bỏ nó ra desktop là xong.

Tham khảo cho URI: [Obsidian URI - Obsidian Help](https://help.obsidian.md/Concepts/Obsidian+URI)
